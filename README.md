<h1 align="left">Hi there! 👋 Welcome</h1>

Senior software engineer and tech lead with 18+ years building distributed systems, cloud infrastructure, and database architecture across OCI, AWS, and GCP. Currently at Oracle, leading multicloud database features for Oracle Database at Google Cloud.

I started coding at 14 with Pascal, which happens to be the name of my son. Never really stopped since. These days I spend a lot of time working with AI agents, using Claude Code, Codex and other tools to plan, build, and ship production code. It has changed how I think about what is possible as an engineer.

I have led teams and projects across multiple countries and industries, from banking infrastructure processing 100M+ transactions a year to cloud-scale APIs serving multiple regions across different continents.

Originally from Punta Arenas, Chile, the southernmost city in Patagonia. Now based in the Seattle area. Husband to Harper and dad to Pascal.
